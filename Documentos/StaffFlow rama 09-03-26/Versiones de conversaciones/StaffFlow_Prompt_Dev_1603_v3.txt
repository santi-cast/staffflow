Tarea 4: ProcesoDiario @Scheduled (PENDIENTE -- siguiente paso)
  Clase: service.scheduled.ProcesoDiario.java
  Cron: "0 55 23 * * *" (23:55 -- cambiado de 00:01, ver D-025)

  DISENO DEFINITIVO (decidido sesion 16, 16/03/2026):

  Un unico metodo @Scheduled a las 23:55 con dos tareas en secuencia:

  TAREA A -- Cierre ausencias injustificadas:
    Obtener todos los empleados activos.
    Para cada empleado sin fichaje en la fecha actual (LocalDate.now()):
      Crear fichaje con tipo AUSENCIA_INJUSTIFICADA.
      Sin horaEntrada ni horaSalida (ausencia sin horas).
      Observaciones automaticas: "Generado automaticamente por ProcesoDiario".
      Usuario de auditoria: terminal_service (usuarioId=4).
    El ENCARGADO puede corregir con E23 al dia siguiente si procede.

  TAREA B -- Procesado planificaciones pendientes:
    Buscar planificacion_ausencias con procesado=false
    y fecha <= LocalDate.now().plusDays(1) (manana).
    Para cada registro:
      Verificar si ya existe fichaje para ese empleado en esa fecha.
      Si no existe: crear fichaje con el TipoFichaje correspondiente
        al TipoAusencia del registro (mapeo enum a enum).
      Marcar planificacion como procesado=true.
    Festivos globales (empleadoId=null): crear fichaje para
      todos los empleados activos de esa fecha.

  Razon del cron 23:55 y fecha <= manana:
    A las 23:55 se cierran las ausencias injustificadas del dia actual
    antes de que acabe el dia. Las planificaciones para manana se
    procesan en el mismo ciclo con fecha <= manana, eliminando la
    necesidad de un segundo @Scheduled a las 00:01.

  Nota primer dia de uso:
    Las ausencias planificadas para el primer dia deben crearse
    manualmente con E22. ProcesoDiario necesita al menos un ciclo
    de 23:55 para procesar planificaciones.

  Nota sobre saldos anuales:
    ProcesoDiario NO toca saldos_anuales. Los fichajes acumulan
    jornadaEfectivaMinutos. ADMIN recalcula saldos con E40 cuando
    lo necesita. El cierre del dia es responsabilidad del ENCARGADO
    usando E22/E23/E25/E36.

  Desviacion asociada: D-025.
