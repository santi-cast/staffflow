TAREA 1 -- CAPA DE RED PRIORIDAD 1
Sesion 27 -- 03/04/2026 -- COMPLETADA
====================================================

Objetivo:
  Crear las interfaces Retrofit y los repositorios para todos los
  endpoints que usan las 6 pantallas de este bloque. Hacerlo en
  una sola tarea evita interrumpir la implementacion de cada
  fragment para crear sus dependencias de red.

Ficheros a crear (data/remote/api/):
  TerminalApiService.kt
    E48 POST /terminal/entrada          -> TerminalEntradaResponse
    E49 POST /terminal/salida           -> TerminalSalidaResponse
    E50 POST /terminal/pausa/iniciar    -> TerminalPausaResponse
    E51 POST /terminal/pausa/finalizar  -> TerminalPausaResponse

  AuthApiService.kt
    E01 POST /auth/login                -> LoginResponse

  SaldoApiService.kt
    E41 GET  /saldos/me                 -> SaldoResponse

  PresenciaApiService.kt
    E35 GET  /presencia/parte-diario    -> ParteDiarioResponse

Ficheros a crear (data/remote/repository/):
  TerminalRepository.kt  <- llama a TerminalApiService
  AuthRepository.kt      <- llama a AuthApiService
  SaldoRepository.kt     <- llama a SaldoApiService
  PresenciaRepository.kt <- llama a PresenciaApiService

Patron de repositorio:
  Metodos suspendibles que devuelven Result<T>.
  El ViewModel consuma Result.onSuccess / Result.onFailure.
  Los errores HTTP se parsean como ErrorResponse via Gson.
